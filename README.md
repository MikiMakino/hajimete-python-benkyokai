# hajimete-python-benkyokai

「はじめてのPython勉強会」で使用する資料・練習問題・サンプルコードをまとめたものです。

Pythonやコードに興味はあるけれど、何から始めたらよいか分からない。
そんな方でも無理なく取り組めるように、学習会で使う内容を分かりやすく整理しています。

実際に手を動かしながらPythonにふれ、業務や日々の工夫につながる第一歩を目指します。

---

## このリポジトリで扱う内容

学習会では、テーマに応じて次のような内容を扱います。

- Python（基本的なプログラミング）
- Python in Excel（ExcelでPythonを使う）
- Office Scripts（TypeScriptでExcelを自動化する）

---

## 対象

この勉強会は、次のような方を対象としています。

- Pythonに興味はあるけれど、まだあまり触ったことがない方
- 業務に活かせるヒントを探している方
- コードを書くことに少し不安がある方
- まずはやさしく試してみたい方

プログラミング経験がなくても大丈夫です。
分からないところがあっても、一つずつ試しながら進めていくことを大切にしています。

---

## 学習会の進め方

学習会では、必要に応じて次のような流れで進めます。

1. テーマの説明を聞く
2. サンプルデータ・コードを確認する
3. 練習問題に取り組む
4. 分からないところを確認する
5. できた内容を共有する

自分のペースで進めながら、気になったことを試してみる時間も大切にしたいと思っています。

### 各フォルダの役割

- `docs/`
  学習会ごとに資料を用意しています。その日の資料は開催日フォルダを確認してください。
  開催日フォルダには、README・練習問題・解答解説・サンプルデータがすべて入っています。

- `exercises/`
  当日やったことを自学・復習できるような練習問題があります。
  カテゴリ・テーマごとに整理しています。

- `samples/`
  サンプルコードをカテゴリ・テーマごとに整理しています。
  まず自分で試してから参照することをおすすめします。

- `images/`
  説明用の画像や図を置く場所です。

## 使い方

### 当日の資料を見る

`docs/` → 開催日フォルダ → テーマフォルダの `README.md` を確認します。

### 自学・復習する

`exercises/` のテーマフォルダにある問題ファイルを読んで、自分でコードを書いてみます。

## 学ぶときの考え方

この勉強会では、最初から完璧に理解することよりも、まずは動かしてみることを大切にしています。

- まずは真似して動かしてみる
- 少しだけ変えてみる
- エラーが出ても慌てずに確認する
- 分からないことを言葉にしてみる

その積み重ねが、Pythonに慣れる一番の近道になると考えています。

## お願い

学習会は、参加者同士が安心して学べる場でありたいと考えています。

- 分からないことを気軽に聞けること
- できないことがあっても大丈夫と思えること
- お互いの学びを尊重できること

そんな雰囲気を大切にしながら進めていきます。
