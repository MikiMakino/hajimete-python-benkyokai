# hajimete-python-benkyokai
「はじめてのPython勉強会」で使用する資料・練習問題・サンプルコードをまとめたものです。  

Pythonやコードに興味はあるけれど、何から始めたらよいか分からない。  
そんな方でも無理なく取り組めるように、学習会で使う内容を分かりやすく整理しています。

実際に手を動かしながらPythonにふれ、業務や日々の工夫につながる第一歩を目指します。

---

## このリポジトリで扱う内容

このリポジトリには、学習会で使う次のような内容をまとめていきます。

- 学習会資料
- 練習問題
- サンプルコード
- 補足メモ
- 参考リンク

初心者の方が「見て終わり」ではなく、実際に手を動かしながら学べるような構成を目指しています。

---

## 対象

この勉強会は、次のような方を対象としています。

- Pythonに興味はあるけれど、まだあまり触ったことがない方
- 業務に活かせるヒントを探している方
- コードを書くことに少し不安がある方
- まずはやさしく試してみたい方

プログラミング経験がなくても大丈夫です。  
分からないところがあっても、一つずつ試しながら進めていくことを大切にしています。

---

## 学習会の進め方

学習会では、必要に応じて次のような流れで進めます。

1. テーマの説明を聞く
2. サンプルコードを動かしてみる
3. 練習問題に取り組む
4. 分からないところを確認する
5. できた内容を共有する

自分のペースで進めながら、気になったことを試してみる時間も大切にしたいと思っています。

## 使い方

### 1. リポジトリを開く

GitHub 上でこのリポジトリを開き、学習会で案内された資料やコードを確認します。

### 2. 必要なファイルを見る

学習会で使う資料は `docs/`、練習問題は `exercises/`、サンプルコードは `samples/` に配置します。

### 3. サンプルコードを試す

サンプルコードは、各自のPC環境やオンライン環境で実行しながら試します。

### 4. 練習問題に取り組む

コードを少し書き換えたり、自分で考えて動かしてみたりしながら学びます。

---

## 学ぶときの考え方

この勉強会では、最初から完璧に理解することよりも、まずは動かしてみることを大切にしています。

- まずは真似して動かしてみる
- 少しだけ変えてみる
- エラーが出ても慌てずに確認する
- 分からないことを言葉にしてみる

その積み重ねが、Pythonに慣れる一番の近道になると考えています。

## お願い

学習会は、参加者同士が安心して学べる場でありたいと考えています。

- 分からないことを気軽に聞けること
- できないことがあっても大丈夫と思えること
- お互いの学びを尊重できること

そんな雰囲気を大切にしながら進めていきます。
