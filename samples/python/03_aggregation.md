# サンプル：集計する

```python
# 商品コードごとの合計金額
整理済み.groupby("商品コード")["金額"].sum()

# カテゴリごとの合計金額
整理済み.groupby("カテゴリ")["金額"].sum()

# 商品コードごとの件数
整理済み.groupby("商品コード")["金額"].count()

# 金額の基本統計
print("合計：", 整理済み["金額"].sum())
print("平均：", 整理済み["金額"].mean())
print("最大：", 整理済み["金額"].max())
print("最小：", 整理済み["金額"].min())
```
