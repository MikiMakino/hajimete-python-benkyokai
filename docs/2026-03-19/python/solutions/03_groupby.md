# 解答3：商品コードごとの合計金額を集計する

## コード

```python
整理済み.groupby("商品コード")["金額"].sum()
```

---

## 考え方

```
整理済み全体 → 商品コードでグループ分け → 各グループの金額を合計
```

`groupby()` は Excel のピボットテーブルに近いイメージです。

---

## かえてみる

カテゴリごとに集計するには？

```python
整理済み.groupby("カテゴリ")["金額"].sum()
```

件数を数えるには？

```python
整理済み.groupby("商品コード")["金額"].count()
```
