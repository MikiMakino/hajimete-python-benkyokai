# 解答4：整理済みデータを Excel ファイルに保存する

## コード

```python
整理済み.to_excel("結果.xlsx", index=False)
```

---

## 考え方

- `to_excel()` で DataFrame を Excel ファイルとして書き出します
- `"結果.xlsx"` の部分は自由にファイル名をつけられます
- `index=False` を指定すると、行番号（0, 1, 2...）が出力されません

---

## 確認

保存後に Excel を開いて、次を確認しましょう。

- [ ] 列名（見出し行）が正しく入っている
- [ ] データが正しく入っている
- [ ] 余分な行番号列がない（`index=False` の効果）
